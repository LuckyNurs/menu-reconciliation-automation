# Menu Reconciliation Automation

Python-based data reconciliation tool to validate menu consistency
between two backend systems (source vs target).

## What It Does
- Fetches active menu data from two databases
- Normalizes and compares menu IDs
- Detects mismatches between systems
- Generates CSV reports per outlet
- Produces a reconciliation summary

## Tech Stack
- Python
- Pandas
- MySQL
- PostgreSQL

## Disclaimer
This project is a **sanitized portfolio version** inspired by a real
production data validation workflow.
All credentials, schemas, endpoints, and business identifiers
have been anonymized.
